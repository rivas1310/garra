🖨️ Impresora Térmica - Original

📋 IMPORTANTE:

Esta página puede abrirse desde cualquier dispositivo.

Si el plugin está en el mismo dispositivo, usa localhost:8000

Si el plugin está en otro dispositivo, usa la IP del teléfono Android (ej: 192.168.1.76:8000)

Si el plugin está en el mismo dispositivo: localhost:8000 | Si está en otro: IP:8000

Obtén la MAC desde el plugin Android en "Ver impresoras disponibles"

⚠️ Si no funciona:

  1. Verifica que el plugin Android esté ejecutándose
  2. Asegúrate de que esté escuchando en el puerto 8000
  3. Verifica que la IP sea correcta
  4. Ambos dispositivos deben estar en la misma red Wi-Fi

📱 Configuración:

  1. Mismo dispositivo: Usa localhost:8000
  2. Dispositivo diferente: Usa la IP del teléfono Android
  3. Obtén la IP desde Configuración > Wi-Fi > Tu red > IP
  4. Ejemplo: 192.168.1.76:8000

✅ Cuando funcione:

Deberías ver un ticket impreso con:

  • Logo de Bazar EnviosPerros
  • Información del negocio
  • Productos de ejemplo
  • Códigos QR y de barras
