🖨️ Impresora Térmica - Bazar EnviosPerros

📋 Instrucciones importantes:

  1. Esta página debe abrirse en el navegador del teléfono Android donde está ejecutándose el plugin
  2. Asegúrate de que el plugin de impresión térmica esté ejecutándose en el puerto 8000
  3. Verifica que tu impresora térmica esté emparejada con el teléfono Android
  4. Ingresa la IP de tu teléfono Android (no localhost)
  5. Ingresa la dirección MAC de tu impresora térmica
  6. Haz clic en "Probar Conexión" para verificar

Obtén la IP desde Configuración > Wi-Fi > Tu red > IP

Obtén la MAC desde el plugin Android en "Ver impresoras disponibles"

⚠️ Posibles errores

failed to fetch: Verifica la IP del teléfono o que el plugin esté ejecutándose

Error de conexión: Verifica que el plugin esté corriendo en el puerto 8000

IP incorrecta: Asegúrate de usar la IP correcta de tu teléfono Android

📱 Cómo obtener la IP de tu teléfono:

  1. Ve a Configuración
  2. Selecciona Wi-Fi
  3. Toca en tu red Wi-Fi conectada
  4. Busca Dirección IP
  5. Copia esa IP (ej: 192.168.1.100)
